# Telegram Weather Bot

Бот погоды для Telegram с запуском на Railway.

Что есть:
- inline-кнопки `Сейчас`, `Сегодня`, `Завтра`, `5 дней`, `Обновить`, `Мой город`
- текущая погода, прогноз на сегодня, завтра и 5 дней
- GIF-анимации для солнца, облаков, дождя, снега, грозы, тумана, ветра и жары
- более умный совет по одежде с учетом ощущаемой температуры, ветра, осадков и перепада за день
- сохранение последнего города или геопозиции в `data/user_places.json`

## Переменные Railway

В Railway открой `Variables` и добавь:

```text
TELEGRAM_BOT_TOKEN=токен_бота_из_BotFather
OPENWEATHER_API_KEY=ключ_из_OpenWeather
```

`.env` в GitHub не загружай.

## Запуск

Railway берет команду из `railway.json`:

```bash
python app.py
```

При первом запросе бот сам создаст GIF-файлы в `assets/weather/` внутри контейнера.

## Проверка

После деплоя в логах Railway должна появиться строка:

```text
Weather bot started
```

Потом напиши боту `/start`, затем город, например `Москва`.
